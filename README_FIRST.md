# MedSci Skills Classroom Installer

This ZIP is for non-programmer classroom installation.

## Windows

1. Unzip this folder.
2. Double-click `installers/install-windows.cmd`.
3. Restart Claude Code Desktop, Codex Desktop, or Cursor.
4. Paste this first test prompt:

```text
MedSci Skills가 설치됐는지 확인하고, 오늘 실습에 쓸 대표 스킬 5개만 보여줘.
```

If Windows shows a security warning, choose **More info** and then **Run anyway**.
The installer copies the `skills/` folder into local agent skill folders and writes
an install log in this folder.

## macOS

1. Unzip this folder.
2. Double-click `installers/install-macos.command`.
3. Restart Claude Code Desktop, Codex Desktop, or Cursor.
4. Paste the same first test prompt above.

If macOS blocks the file, right-click it, choose **Open**, and confirm once.
