# American Journal of Roentgenology (AJR) — Reviewer Profile

## Submission System
- **Platform:** Editorial Manager
- **Review type:** Single-blind (reviewer sees author identity)
- **Publisher:** ARRS (American Roentgen Ray Society)

## Required Structure

Section-by-section review with continuous numbering across sections:

1. Title
2. Abstract
3. Introduction
4. Methods
5. Results
6. Discussion
7. References
8. Tables
9. Figures

Each section lists numbered comments; numbering continues across sections (not reset per section).

## Scorecard

AJR does not expose a rigid numeric scorecard; confidential comments to editor carry the recommendation rationale.

## Recommendation Options

- Accept
- Minor Revision
- Major Revision
- Reject (resubmission allowed)
- Reject
